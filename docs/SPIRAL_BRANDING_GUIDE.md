# Spiral - Visual Branding Guide

## Logo Design Specifications

### Primary Logo Concept: The Fingerprint Spiral

**Core Concept:**
A spiral that doubles as a fingerprint/thumbprint, representing both the hypnotic nature of doom scrolling and the personal, human element of the intervention.

### Version 3: Dynamic Spiral (Animated State)
```
Animation stages:

Stage 1: Idle (breathing)
- Gentle 360¬∞ rotation (30 second loop)
- Subtle scale pulse (0.95x to 1.05x)
- Glow effect pulses in sync

Stage 2: Hypnotic (detection in progress)
- Rapid clockwise rotation (2 seconds per rotation)
- Intensifying glow
- Slight color shift (blue ‚Üí purple)

Stage 3: Breaking (intervention triggered)
- Spiral fragments into pieces
- Each segment flies outward
- Fragments fade to 0% opacity
- Duration: 0.8 seconds

Stage 4: Reformed (success/completion)
- Fragments reassemble from edges
- Color shifts to success green (#10B981)
- Gentle pulse once
- Returns to idle state
```

**Usage:**
- Home screen widget (idle)
- During detection (hypnotic)
- Intervention screen (breaking)
- Post-reflection (reformed)

## Animation Principles

### Timing Functions
- **Ease Out:** Default for entrances and scale ups
- **Ease In:** Exits and scale downs
- **Spring:** Interactive elements (buttons, toggles)
- **Linear:** Progress indicators, loading

### Duration Guidelines
- **Micro:** 0.1-0.2s - Hover states, small changes
- **Short:** 0.3-0.4s - Button presses, toggles
- **Medium:** 0.5-0.7s - View transitions, modal presentations
- **Long:** 0.8-1.2s - Complex animations, celebrations
- **Extra Long:** 2-3s - Ambient animations (spiral breathing)

### Animation Catalog

**Fade In**
```
Duration: 0.3s
Timing: Ease out
Opacity: 0 ‚Üí 1
```

**Slide Up**
```
Duration: 0.4s
Timing: Spring (damping: 0.8)
Transform: translateY(100%) ‚Üí translateY(0)
Opacity: 0 ‚Üí 1
```

**Scale Bounce**
```
Duration: 0.5s
Timing: Spring (damping: 0.6, response: 0.3)
Transform: scale(0.8) ‚Üí scale(1.05) ‚Üí scale(1)
```

**Shake** (Error feedback)
```
Duration: 0.4s
Timing: Linear
Transform: translateX(-10) ‚Üí translateX(10) ‚Üí [repeat 3x] ‚Üí translateX(0)
```

## Motion Design

### Signature Animations

**1. Spiral Breathing (Idle State)**
```
Loop: Infinite, 30 second cycle
Keyframes:
  0%: Scale 1.0, Rotation 0¬∞, Opacity 100%
  25%: Scale 1.05, Rotation 90¬∞, Opacity 90%
  50%: Scale 1.0, Rotation 180¬∞, Opacity 100%
  75%: Scale 0.95, Rotation 270¬∞, Opacity 90%
  100%: Scale 1.0, Rotation 360¬∞, Opacity 100%
Easing: Ease-in-out throughout
```

**2. Spiral Hypnotic (Detection Active)**
```
Loop: Continuous until intervention
Duration: 2s per rotation
Rotation: 0¬∞ ‚Üí 360¬∞ (clockwise)
Scale: Subtle pulse 0.98 ‚Üí 1.02
Glow: Intensity increases over time
Color: Shifts from blue to purple gradually
Easing: Linear rotation, ease-in-out pulse
```

**3. Spiral Breaking (Intervention Trigger)**
```
Duration: 0.8s total
Phases:
  Phase 1 (0-0.3s): Spiral speeds up, vibrates
  Phase 2 (0.3-0.5s): Fragments into 8-12 pieces
  Phase 3 (0.5-0.8s): Pieces fly outward radially, fade out

Fragment behavior:
- Each piece rotates independently
- Random slight rotation variation
- Fade to 0% opacity
- Scale down 0.9x as they fade
Easing: Ease-in for acceleration, ease-out for dispersion
```

**4. Spiral Reformed (Success)**
```
Duration: 1.0s total
Phases:
  Phase 1 (0-0.4s): Fragments fly in from edges
  Phase 2 (0.4-0.7s): Pieces snap into place
  Phase 3 (0.7-1.0s): Color shift to green, pulse once

Color transition:
- Blue ‚Üí Green over 0.3s
- Single scale pulse: 1.0 ‚Üí 1.15 ‚Üí 1.0
Easing: Spring animation with slight overshoot
```

## Accessibility

### Color Contrast Ratios
All text meets WCAG AA standards:
- Normal text: 4.5:1 minimum
- Large text (18pt+): 3:0 minimum
- UI components: 3:1 minimum

### Tested Combinations
‚úÖ White on Deep Purple: 12.5:1 (AAA)
‚úÖ Electric Blue on Deep Purple: 4.8:1 (AA)
‚úÖ White on Alert Red: 4.6:1 (AA)
‚úÖ Black on Soft Gray: 19.8:1 (AAA)

### Reduce Motion
When `reduceMotion` is enabled:
- Spiral: Static or slow fade instead of rotation
- Intervention: Fade in instead of break animation
- All transitions: Crossfade instead of slide/scale
- Haptics: Still fire (not motion-based)

### VoiceOver
- All interactive elements: Labeled
- Spiral animation: "Spiral animation, currently idle"
- Intervention: Announces immediately
- Stats: Read values with context

## Brand Guidelines Summary

### Do's ‚úÖ
- Use gradient backgrounds liberally
- Maintain spacing and hierarchy
- Keep animations smooth and purposeful
- Use bold, confident copy
- Respect safe areas and margins
- Test on actual devices

### Don'ts ‚ùå
- Don't rotate logo (except in animation)
- Don't use unapproved colors
- Don't add drop shadows to logo
- Don't stretch or distort logo
- Don't use busy backgrounds behind text
- Don't use low-contrast color combinations
